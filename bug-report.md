# Bug Report

## Issue Title
No validation when Start Date is later than End Date

## Severity
Medium

## Steps to Reproduce

1. Open the random date generator page
2. Enter a start date later than the end date
3. Click Generate

## Expected Result

The application should display an error message indicating that the start date must be earlier than the end date.

## Actual Result

The tool does not clearly prevent the incorrect input and may still attempt to generate results.

## Impact

This could lead to invalid or unexpected date outputs.
